## Keyboard Navigation

```
Tab Navigation:
- Tab: Move between elements
- Shift+Tab: Move backward
- Enter: Activate button/expand row
- Space: Toggle checkbox/expand row
- Escape: Close modal

Shortcuts (Optional):
- Alt+A: Approve current line
- Alt+E: Request edit
- Alt+F: Flag line
- Alt+C: Add comment
```
